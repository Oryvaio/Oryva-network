# ORYVA

> Lock. Borrow. Repay.

The first protocol designed to unlock liquidity from locked Pump.fun tokens on Solana. Oryva enables token holders to borrow SOL against their locked positions without selling their tokens, allowing users to access capital while maintaining exposure to their assets.

Built for the next generation of on-chain liquidity, Oryva bridges the gap between token ownership and capital efficiency.

---

## 🚀 What is Oryva?

Oryva is a decentralized borrowing protocol that allows users to use locked Pump.fun tokens as collateral and receive SOL liquidity instantly.

Instead of choosing between holding tokens or accessing capital, users can do both.

```text
Lock Tokens
      ↓
Borrow SOL
      ↓
Use Capital
      ↓
Repay Loan
      ↓
Unlock Tokens
```

The protocol introduces a new primitive for Solana DeFi by transforming illiquid token positions into productive capital. Similar collateralized lending models on Solana allow users to borrow against assets while retaining ownership exposure. :contentReference[oaicite:0]{index=0}

## 🏦 How It Works

### Step 1 — Lock

Deposit eligible Pump.fun tokens into Oryva.

### Step 2 — Borrow

Receive SOL based on collateral value.

### Step 3 — Utilize

Use borrowed SOL for trading, liquidity, farming, or ecosystem participation.

### Step 4 — Repay

Repay the loan plus protocol fees.

### Step 5 — Unlock

Retrieve collateralized tokens.
